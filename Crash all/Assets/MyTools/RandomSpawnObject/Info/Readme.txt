Данный инструмент можно использовать для спавна объектов в контролируемой зоне. Например выпадение монеток

Vector2 minMaxRadius Задает радиус в котором будет выбираться рандомная точка для спавна. 
Min радиус внутренний радиус который игнорируется, таким образом можно задать определенный радиус спавна

public void Spawn(Vector3 position, Vector3 positionDrop, Action onFinish)
position - стартовая позиция
positionDrop - позиция где будет выбираться рандомная точка из радиуса
onFinish - делегат, сообщает когда спавн закончен

У скрипта можно включить debug и увидеть радиус спавна
