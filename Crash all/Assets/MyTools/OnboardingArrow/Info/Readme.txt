Для работы со стрелкой необходимо инстаинтировать префаб и использовать метод Init для передачи стартовой позиции и конечной.

Для уничтожения стрелки необходимо вызвать метод Stop();

Для корректного размера стрелки нужно настроить скейл по Z и скейл по X через параметр sizeXMultiplier
Высота ставится через posY
